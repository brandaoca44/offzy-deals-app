# 🚀 Offzy - Plataforma de Ofertas

Offzy é uma aplicação web inspirada em plataformas como Pelando e Promobit, onde usuários podem descobrir, compartilhar e avaliar ofertas.

O projeto foi desenvolvido com foco em experiência de usuário e arquitetura moderna em React.

---

## ✨ Funcionalidades

- Feed de ofertas com scroll infinito
- Sistema de votos (👍 / 👎)
- Cálculo de desconto automático
- Filtro por categorias
- Busca de produtos
- Publicação de novas ofertas
- Bloqueio de ofertas para usuários não logados
- Persistência com localStorage
- Layout responsivo (desktop e mobile)

---

## 🛠️ Tecnologias

- React
- JavaScript (ES6+)
- Vite
- React Router
- CSS (variáveis e responsividade)

Este projeto foi desenvolvido como parte do meu portfólio com o objetivo de demonstrar habilidades em desenvolvimento front-end utilizando React.

Principais pontos abordados:

- Estruturação e organização de código em React
